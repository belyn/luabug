
# Luabug
1. author:marvin  
1. **luabug** 是基于vscode开发的一款用于lua语言开发者使用的插件, 
2. 由于luaIde在v0.3.7之后不进行开源，本插件只是基于luaIde V0.3.7升级维护
3. 有兴趣查看原作者的请点这(https://github.com/k0204/LuaIde)
